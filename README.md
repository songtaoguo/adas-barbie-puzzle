# 🎀 Ada's Puzzle

A sparkly drag-and-drop jigsaw puzzle game for Ada.

## How to play
1. Pick a puzzle from the home screen
2. Drag pieces from the side rails onto the board
3. When you put a piece near its correct spot, it snaps in!
4. Stuck? Hit **💡 Hint** — the finished picture shows for 5 seconds
5. Finish the puzzle and watch the princesses dance 💃

## Levels
- Start with simple 3×3 puzzles
- Work up to 5×5 puzzles with more pieces

## Built with
- Plain HTML, CSS, JavaScript — no frameworks, no build step
- All puzzle art hand-drawn as SVG
- Win celebration uses Web Audio API (no audio files)
